#  重要管理 安全(Security)
```
機密性 (Condfidentiality) = 留下紀錄
鑑別性 (Authenticity)     = 確認身分
不可否認性 (Non-repudiation) = 確保資了的真實性和可追溯性
完整性(Intrgrity) = 確保資料的完整性
可歸責性 (Accountability) = 確認事件的源頭
可用性(Availability)  = 確保系統持續運作
可靠度 (Reliability)  = 確保系統穩定信和可信度
```
